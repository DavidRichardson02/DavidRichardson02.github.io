FPGA Signal Control System

A real-time sensing and control framework on a Nexys A7 FPGA integrating: ToF distance mapping, PWM fan control, UART telemetry streaming, VGA HUD overlays, and PIR-based motion detection.

Architecture

  • ToF sensor engine with I²C master, IRQ handling, and sample cadence control.
  • UART packetizer encoding timestamped distance and angle data with status/CRC.
  • VGA range-plot framebuffer plus HUD overlay rendering temperature, fan duty, PIR status, and UART activity.
  • PWM fan control loop using Q1.15 fixed-point temperature measurements.

Key Engineering Decisions

  • CDC-safe interfaces between 100 MHz system clock and 25 MHz pixel clock domains.
  • Use of opcode-style documentation and hierarchical Verilog modules to keep the design maintainable.
  • Integration with WaveForms and MATLAB for correlated digital/analog debugging.

Generic Quadtree Barnes–Hut N-Body Simulator

A 2D interactive N-body simulation engine built in modern C++17 using a quadtree spatial partitioning scheme and the Barnes–Hut approximation to support large systems in real time.

Core Ideas

  • Adaptive quadtree subdivision based on particle distribution and depth limits.
  • Force aggregation using center-of-mass approximations for distant cells.
  • Careful timestep integration decisions to balance stability and responsiveness.

Implementation Details

  • Template-based quadtree structure with configurable force law and thresholds.
  • Use of Eigen/GLM-style math primitives and standard containers.
  • Visualization loop designed for real-time interaction and debugging.

Serial Hashed-Octree Barnes–Hut 3D Simulator

A 3D extension of the Barnes–Hut simulation concept using a Morton-ordered hashed octree to support better memory locality and efficient 3D force evaluation.

Architecture

  • Morton-coded (Z-order) indexing for octree nodes to improve cache behavior.
  • Hashed storage of octree nodes rather than purely pointer-based trees.
  • 3D center-of-mass aggregation and angle-based criteria for approximation.

Design Notes

  • Focus on clear serial design first, with an eye toward future parallelization.
  • Low-level profiling to understand memory access patterns and hotspots.

Automated CSV Dataset Analysis & Modeling

A modular C++17 analytics engine for structured scientific datasets, focusing on schema inference, standardization, diagnostics, and modeling for CSV-based workflows.

Key Components

  • Custom tokenization and parsing pipeline resilient to malformed rows and mixed types.
  • Columnar data representation to support efficient scanning, transforms, and statistics.
  • Hooks for visualization and model-fitting on top of the standardized data structures.

Use Cases

  • Rapid exploration of experimental logs and simulation outputs.
  • Repeatable data-cleaning and feature extraction pipelines for later analysis.
